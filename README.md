# Discord_groupChat
This project is a **Realtime Chat Application** built using React (Frontend) and Spring Boot with WebSocket (Backend). It allows multiple users to join chat rooms and exchange messages in real time using STOMP over SockJS.

- The application supports:
- Creating or joining chat rooms
- Real-time message delivery without page refresh
- Persistent message storage (MongoDB)
- REST APIs for loading chat history
- WebSocket-based live communication
